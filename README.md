## DESN368: Week 8 - CSS Micro-Challenges & Lecture Materials

Overview

This package contains the lecture slides and the interactive lab assignment for Week 03. The focus is on three core "Level Up" skills for web design:

CSS Variables (Theming & Consistency)

HTML5 Media (Video & Audio attributes)

Motion (Transitions, Transforms, & Animation)

Files Included

1. Lecture Slides

css_level_up.html: The standard slide deck covering the basics.

css_deep_dive.html: An extended slide deck with advanced examples (Object Fit, Keyframes, Accessibility).

Usage: Double-click to open in Chrome. Use F11 for full-screen presentation.

2. The Lab Assignment

interactive_css_lab.html: A single-file, interactive workbench containing 10 Micro-Challenges.

Instructions for Students

How to Run the Lab:

Download interactive_css_lab.html to your computer.

Open the file in Google Chrome (or Firefox/Safari).

The file runs locallyâ€”no internet connection or server is strictly required (though image assets are pulled from the web).

How to Complete the Assignment:

Scroll through the 10 challenges.

Read the prompt on the left (e.g., "Update this button to use the brand color variable").

Type the correct CSS code into the Code Editor box.

Click Check Code.

If correct, the preview updates, and the progress bar advances.

If incorrect, the box turns red.

Stuck? Click "Need a hint?" for syntax help.

Instructor / Deployment Notes

To Share via Canvas:
Upload interactive_css_lab.html to the Assignment or Module. Instruct students to download it rather than previewing it inside Canvas (Canvas previews often block the JavaScript needed for the interactivity).

To Share via Netlify:

Put interactive_css_lab.html into a folder named week-03-lab.

Rename the file to index.html.

Drag the folder to app.netlify.com/drop.

Share the generated link.
